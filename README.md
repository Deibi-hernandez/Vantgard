# Vantgard
Vantgard redefine la cava personal en Antofagasta. Somos una boutique de destilados de autor y etiquetas de colección, donde la tradición se encuentra con la vanguardia. Cada botella es una pieza de arte seleccionada para paladares que exigen exclusividad, origen y maestría. Descubre el grado más alto de sofisticación. Vantgard: Curaduría Líquida.

## Backend Django (Liquor_Store)

Ruta del proyecto:

`/home/runner/work/Vantgard/Vantgard/Vantgard/Liquor_Store`

### Dependencias

Instalar con:

```bash
python -m pip install -r /home/runner/work/Vantgard/Vantgard/Vantgard/Liquor_Store/requirements.txt
```

### Setup local rápido

```bash
cd /home/runner/work/Vantgard/Vantgard/Vantgard/Liquor_Store
python manage.py migrate
python manage.py runserver
```

### Pruebas locales

```bash
cd /home/runner/work/Vantgard/Vantgard/Vantgard/Liquor_Store
python manage.py check
python manage.py test
```
